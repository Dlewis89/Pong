# <p align="center">Pong</p>

Implementation of the game Pong in python using the Turtle module 

## Build

For instructions on building the project please see [BUILD.md](./docs/BUILD.md) for more.

## Controls

Left Paddle
- Up = W Key
- Down = S Key

Right Paddle
- Up = Up Arrow Key
- Down = Down Arrow Key
