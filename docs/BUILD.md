# <p align="center">Instructions</p>

## Dependencies
- Python3

## Running the game

```sh 
$ git clone https://github.com/Dlewis89/pong.git
$ cd pong
```

Depending on your OS you may need to run a slightly different command to start the game.

- Windows
    ```sh
    $ python main.py
    ```
- Linux
    ```sh
    $ python3 main.py
    ```
- MacOs
    ```sh
    $ python3 main.py
    ```
